# Employee Management Service

Spring Boot REST API for managing employees, projects, and assignments.

## Features
- Employee CRUD operations
- Project management
- Assignment handling
- Validation
- Security (RBAC)
- DTO layer

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL

## How to Run
1. Clone repo
2. Run application
3. Access APIs via Postman
